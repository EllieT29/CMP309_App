# CMP309_App
CMP309 App development for mobile devices

This app is called 'Bloom' it is a wellness app that encourages a connection with nature. 

## Tasks
Bloom provides 5 wellness tasks each day, once a task has been complete it can be checked off in the 'Tasks' screen.
Each time a task has been complete, the flower image grows a coloured petal and a notfification appears indicating the next task. 
Once all tasks are complete, the flower has 'bloomed' and a notification will appear. 
The flower image can be found on the 'Home' screen and the 'Tasks' screen.

## Journal
Bloom also provides a journal feature to record any thoughts or feelings, you can type in your entry using the bar at the top of the 'Journal' screen, then simply click 'Add' to add your entry. 
Entries can also be deleted. 

## Meditation
On the 'Home' screen a meditation track is displayed, the track can be played using the play icon, paused using the pause icon and stopped using the stop icon. 
The meditation track talks through a meditation session. 

## Daily Quote
On the bottom of the 'Home' screen a daily quote can be found. This updates everyday and requires the phone to be connected to the internet. 
If not connected to the internet then a message will be displayed to state this.
